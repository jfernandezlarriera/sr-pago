## Requesitos

* Docker
* Docker Compose

## Instrucciones

### `npm install`

Levantar la aplicación:

### `docker-compose -f "docker-compose.yml" up -d --build`

Puedes abrir la aplicacion entrando a [localhost](http://localhost) en tu explorador.

Comprueba que levantaste bien la aplicación.
